Powerful Intelligence, Simple Integration.

Our open-source Python SDK is designed to fit into any existing agentic framework. Get started with just a few lines of code.


from kairos_agent_sdk import KairosClient

# Initialize the client
kairos = KairosClient(api_key="kai_...")

# Get a real-time strategy
strategy = kairos.get_strategy(context={...})

# Record the outcome to close the loop
kairos.record_outcome(
    interaction_id=strategy.interaction_id,
    metrics={"booking_confirmed": True}
)
